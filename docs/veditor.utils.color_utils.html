veditor.utils.color_utils.generate_color_series(color, variation, diff=10, reverse=False)[source]¶

Generate light and dark color series.

Parameters

  • color (tuple) – Color [0,255]

  • variation (int) – How many colors to create.

  • diff (int) – How much to change

  • reverse (bool) – If True, sort in descending order.

Returns

colors.

colors (list)

Examples
